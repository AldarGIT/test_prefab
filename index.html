My Excalidraw Library

A collection of prefabs and components for your Excalidraw drawings. Click the button below to open Excalidraw with this library loaded.

Open in Excalidraw

This will open Excalidraw and automatically add your custom library to the libraries panel. You can then access your prefabs from the library icon in the toolbar.


Troubleshooting: If you get an "Invalid library URL" error, make sure:
• GitHub Pages is enabled in your repository settings
• The library.json file is accessible at: https://aldargit.github.io/test_prefab/library.json
• Try the direct import option below as an alternative


Alternative: You can also manually import by copying this URL and pasting it in Excalidraw's library panel: https://aldargit.github.io/test_prefab/library.json
